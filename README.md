# Derek's Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Navigation Menu**: Easy navigation between sections
- **Hero Section**: Eye-catching introduction with call-to-action
- **Projects Showcase**: Display your featured projects with descriptions, technologies, and links
- **Resume Section**: Highlight your experience, skills, and education
- **Contact Section**: Direct links to email, LinkedIn, and GitHub
- **Smooth Scrolling**: Smooth navigation between sections
- **Modern Styling**: Clean, professional design with gradients and hover effects

## Getting Started

### Customization

1. **Update Personal Information**
   - Open `index.html` and replace placeholder text with your information
   - Update your name, title, bio, and contact details
   - Add your email address in the contact section

2. **Add Your Projects**
   - Edit the project cards in the "Featured Projects" section
   - Add project title, description, technologies used, and links
   - Replace placeholder images with your project screenshots

3. **Update Resume Section**
   - Add your work experience and achievements
   - List your technical skills
   - Upload your resume PDF and update the download link
   - Update the path in `href="path/to/your/resume.pdf"`

4. **Customize Styling**
   - Modify colors in `styles.css` (check the `:root` CSS variables)
   - Adjust fonts, spacing, and layout as needed
   - Change color schemes by updating CSS gradient values

## Deployment

This repository is set up for GitHub Pages. Any changes pushed to the `main` branch will be automatically deployed to your GitHub Pages site.

### Important Links to Update

In `index.html`, update these placeholders:

- **Contact Email**: Replace `your.email@example.com` with your actual email
- **LinkedIn**: Replace `https://www.linkedin.com/in/your-profile/` with your LinkedIn URL
- **GitHub**: The GitHub link already points to `dcrock03`
- **Resume PDF**: Replace `path/to/your/resume.pdf` with the actual path to your PDF

## Color Scheme

The default colors are:
- Primary: `#2c3e50` (Dark blue-gray)
- Accent: `#3498db` (Blue)
- Light Background: `#ecf0f1` (Light gray)

Feel free to customize these in the CSS!
